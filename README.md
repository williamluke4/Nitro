## NPM

### Install and Run Dev Server

```bash
git clone https://github.com/williamluke4/nitro.git
cd nitro

npm install
npm run dev
```

### Production Build and Start

```bash
  npm run build
  npm run start
```

## Yarn

### Install and Run Dev Server

```bash
git clone https://github.com/williamluke4/nitro.git
cd nitro

yarn
yarn dev
```

### Production Build and Start

```bash
yarn build
yarn start
```
